# MOS
WEB OS
类似Win10的WEB OS

UI使用[WIN10-UI](http://win10ui.yuri2.cn/src/demo.php) v1.1.2.4

[演示](http://micos.000webhostapp.com/) admin  0000

请用谷歌浏览器,获得最佳浏览效果。

集成组件：

1.layui-layer v3.0.3

2.font-awesome v4.7.0

3.jquery v3.3.1

4.animate v3.6.0

5.ztree v3.5
